What Is Frost Forecast?

FrostForecast is a tool that provides frost prediction alerts for farmers and producers. Using multiple data sources as inputs to a frost prediction model, it will deliver early frost warnings so farmers can focus their control effort.

Get early alerts on frosts events

A machine learning model uses data from different sources to predict frost events and generate early alerts.

Improve decision-making for frost prevention

It helps agricultural producers make informed decisions on active frost control.
