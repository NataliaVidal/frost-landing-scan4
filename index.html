Take Care Of the Crops

FrostForecast is a frost prediction system, thats allows to monitor the climate state of crops.

Find Out More

What Is FrostForecast?

Frost Forecast proposes a precision agriculture system that allows to visualize the climatic state of a crop providing a detailed vision of the state of the crops.

Go to Dashboard

What Is It For?

Using multiple data sources as inputs to a machine learning engine, it will deliver early frost warnings so farmers can focus their control effort.

Improve decision-making for frost prevention

It helps agricultural producers make informed decisions on active frost control.

Get early alerts on frosts

A predictive model uses data from different sources to generate early frost alerts.

Know what’s happening in the state

Sensors are installed in different sectors of the state, providing climatic information about each one of them.

How Does It Works?

Using a network of sensors through a mesh topology, nodes are located at a certain distance in order to capture environmental variables, such as temperature and humidity. A gateway is configured to receive and handle the captured data. These are stored in a database and can be viewed through a dashboard.
